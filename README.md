# HR Analytics Dashboard – Power BI Project

## 📌 Project Overview
The HR Analytics Dashboard is a Power BI project developed to analyze employee data and identify important workforce trends such as employee attrition, salary distribution, job satisfaction, and work-life balance.

This dashboard helps HR departments and business managers make better decisions using data visualization and business intelligence techniques.

---

# 🎯 Objectives
- Analyze employee attrition trends
- Identify departments with high turnover
- Study employee satisfaction levels
- Compare salary distribution across job roles
- Understand the impact of overtime on attrition
- Improve workforce planning and employee retention

---

# 📂 Dataset Information
The dataset contains HR-related employee records including:

- Employee demographics
- Department details
- Job roles
- Monthly income
- Work-life balance
- Job satisfaction
- Attrition status
- Overtime details
- Performance ratings

---

# 🛠 Tools & Technologies Used
- Power BI
- Power Query
- DAX (Data Analysis Expressions)
- CSV Dataset

---

# 📊 Dashboard Features

## KPI Cards
- Total Employees
- Attrition Count
- Attrition Rate
- Average Salary
- Average Years at Company

## Visualizations
- Attrition by Department
- Attrition by Gender
- Salary Distribution
- Employee Age Analysis
- Job Satisfaction Analysis
- Overtime vs Attrition
- Work-Life Balance Insights

---

# 📈 Key Insights
- Employees working overtime showed higher attrition rates.
- Some departments experienced more employee turnover than others.
- Lower job satisfaction increased the chances of attrition.
- Employees with fewer years at the company were more likely to leave.
- Salary distribution varied significantly across different job roles.

---

# 🔄 Data Processing Steps
1. Imported CSV dataset into Power BI
2. Cleaned missing and duplicate values
3. Transformed data using Power Query
4. Created calculated columns and measures using DAX
5. Built interactive dashboard visualizations
6. Added filters and slicers for better analysis
